Sobre mim

Meu nome é Kauã, sou aluno da Escola Germinare TECH do Instituto J&F.

Gosto muito de programação, explorar e treinar novos modos de desenvolver um código, jogar jogos, assitir anime, ouvir música geek.

Sempre tive vontade de querer fazer jogos, mas não sabia programar, nem lógica de programação. Antes de entrar na Germinare tentei fazer um jogo na Unity com ajuda do ChatGPT, que disponibiliza apenas C# como linguagem de programação, mas era muito complexo algumas funções e interfaces, o lado positivo foi o aprendizado de comportamentos básicos de componentes de um jogo.

No curso de Python, aprendi a lógica de programação, o que ajudou a otimizar alguns códigos do jogo. Depois disso parei de utilizar a Unity por um tempo, e mudei para o GameMaker que tinha uma linguagem de programação própria, mas com ferramentas que facilitava o uso, além de estar em português. A linguagem é parecida com a combinação de JavaScript, com Python.

Com as aulas de POO (Programação orientada a objeto) melhorei muito a organização de funções no jogo, principalmente para otimizar o jogo.


Sobre o projeto do meu jogo

A principio o jogo é estilo plataforma 2D, em pixel art, com mecanicas de andar, pular, dash e correr.

Até a presente data de 17/01/2026, o projeto está em desenvolvimento.
Sendo por enquanto um teste, mas se o desenvolvimento for algo grande e promissor, possa ser algo oficial para se lançar como jogo.
